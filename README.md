
# 🛒 Retail Analysis with Walmart Data


## 📌 Project Overview
This project focuses on analyzing Walmart retail sales data using Python Data Analytics tools.  
The main goal is to extract business insights and visualize retail trends.

---

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Features
✔ Data Cleaning  
✔ Data Analysis  
✔ Data Visualization  
✔ Retail Sales Insights  
✔ Customer Behaviour Analysis  

---

## 📂 Dataset
The dataset contains Walmart retail sales information including:
- Sales
- Profit
- Customer details
- Product categories
- Region-wise data

---

## 📈 Analysis Performed
- Monthly Sales Analysis
- Profit Analysis
- Category-wise Performance
- Visualization Charts
- Business Insights

---

## 💡 Business Insights
- High sales categories identified
- Customer purchasing trends analyzed
- Region-wise performance compared
